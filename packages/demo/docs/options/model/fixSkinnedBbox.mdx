Sometimes, some rigged model has the wrong bounding box that when displaying on three.js (usually converted glTF model from [Sketchfab](https://sketchfab.com/))
Enabling this option can resolve that issue by recalculating bounding box size with the influence of the skeleton weight.

## Example
### fixSkinnedBbox: false (default)
<View3D
  src="/egjs-view3d/model/meshopt/trex.glb"
  fixSkinnedBbox={false}
  showBbox
  showExampleCode />

### fixSkinnedBbox: true
<View3D
  src="/egjs-view3d/model/meshopt/trex.glb"
  fixSkinnedBbox={true}
  showBbox
  showExampleCode />
