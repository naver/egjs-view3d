Camera Controls

OrbitControl

View3D provides OrbitControl to control the camera's yaw, pitch, and distance from camera's pivot.
See RotateControl, TranslateControl, and DistanceControl for further information.

import View3D, { OrbitControl } from "@egjs/view3d";

// Create View3D
const view3d = new View3D("#view3d-canvas");

// Add OrbitControl
view3d.controller.add(new OrbitControl({
  // RotateControl's options
  rotate: {
    scale: 1,
  },
  // TranslateControl's options
  translate: {
    scaleToElement: true,
  },
  // DistanceControl's options
  distance: {
    range: {
      min: 0,
      max: 500,
    }
  }
}));

Autoplay

AutoControl can rotate the camera's yaw automatically.
See its options for further information.

import View3D, { AutoControl } from "@egjs/view3d";

// Create View3D
const view3d = new View3D("#view3d-canvas");

// Add AutoControl
view3d.controller.add(new AutoControl({
  speed: 1,
  delay: 2000,
  delayOnMouseLeave: 0,
  pauseOnHover: false,
  canInterrupt: true,
  disableOnInterrupt: false,
}));
