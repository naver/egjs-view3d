
### scaleToElement {#scaleToElement}

<div className="bulma-tags">

</div>

Scale control to fit element size.<br />When this is true, camera's pivot change will correspond same amount you've dragged.

**Default**: true

```ts
import View3D, { TranslateControl } from "@egjs/view3d";
const view3d = new View3D("#view3d-canvas");
const translateControl = new TranslateControl();
translateControl.scaleToElement = true;
view3d.controller.add(translateControl);
view3d.resize();
```
