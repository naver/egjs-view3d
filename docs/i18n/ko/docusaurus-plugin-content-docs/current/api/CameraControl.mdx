
|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|element||HTMLElement that this control is attatched.|
|enabled||Whether this control is enabled or not.|
|setElement||Set HTMLElement to attach this control.|
|destroy||Destroy this control.|
|resize||Resize the control. Usually, the element size is used for parameter.|
|sync||Synchronize internal state with the given camera.|
|update||Update internal state by delta time, and apply it to the given camera.|
|enable||Enable this control.|
|disable||Disable this control.|
