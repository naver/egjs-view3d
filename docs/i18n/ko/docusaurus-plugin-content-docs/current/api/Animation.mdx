## Events
### progress {#event-progress}

<div className="bulma-tags">

</div>

Fires for every animation frame when animation is active.

**Type**: object

|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|event|object|Event object.|
|event.progress|number|Current animation progress value.<br />Value is ranged from 0(start) to 1(end).|
|event.easedProgress|number|Eased progress value.|

### loop {#event-loop}

<div className="bulma-tags">

</div>

Fires for every animation loop except for the last loop<br />This will be triggered only when repeat > 0

**Type**: object

|PROPERTY|TYPE|DESCRIPTION|
|:---:|:---:|:---:|
|event|object|Event object.|
|event.progress|number|Current animation progress value.<br />Value is ranged from 0(start) to 1(end).|
|event.easedProgress|number|Eased progress value.|
|event.loopIndex|number|Index of the current loop.|

### finish {#event-finish}

<div className="bulma-tags">

</div>

Fires when animation ends.

**Type**: void
