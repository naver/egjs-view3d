### initialBbox {#initialBbox}

<div className="bulma-tags">

### size {#size}

<div className="bulma-tags">

</div>

Model's bounding box size<br />Changing this will scale the model.

**Type**: number

```ts
import { GLTFLoader } from "@egjs/view3d";
new GLTFLoader().load(URL_TO_GLTF)
 .then(model => {
   model.size = 100;
 })
```

### fixSkinnedBbox {#fixSkinnedBbox}

<div className="bulma-tags">

</div>

Whether to apply inference from skeleton when calculating bounding box<br />This can fix some models with skinned mesh when it has wrong bounding box

**Type**: boolean

### originalSize {#originalSize}

<div className="bulma-tags">

</div>

Return the model's original bbox size before applying any transform

**Type**: number
