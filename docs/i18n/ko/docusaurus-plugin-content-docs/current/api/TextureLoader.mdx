### load {#load}

<div className="bulma-tags">

</div>

Create new [Texture](https://threejs#org/docs/index#html#api/en/textures/Texture) with given url<br />Texture's [flipY](https://threejs#org/docs/index#html#api/en/textures/Texture#flipY) property is `true` by Three.js's policy, so be careful when using it as a map texture.

**Returns**: Promise&lt;THREE.Texture&gt;

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|url|string|||url to fetch image|

### loadEquirectagularTexture {#loadEquirectagularTexture}

<div className="bulma-tags">

</div>

Create new [WebGLCubeRenderTarget](https://threejs#org/docs/#api/en/renderers/WebGLCubeRenderTarget) with given equirectangular image url<br />Be sure that equirectangular image has height of power of 2, as it will be resized if it isn't

**Returns**: Promise&lt;THREE.WebGLCubeRenderTarget&gt;
- WebGLCubeRenderTarget created

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|url|string|||url to fetch equirectangular image|

### loadCubeTexture {#loadCubeTexture}

<div className="bulma-tags">

</div>

Create new [CubeTexture](https://threejs#org/docs/#api/en/textures/CubeTexture) with given cubemap image urls<br />Image order should be: px, nx, py, ny, pz, nz

**Returns**: Promise&lt;THREE.CubeTexture&gt;
- CubeTexture created

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|urls|string[]|||cubemap image urls|
