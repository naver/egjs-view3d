### display {#display}

<div className="bulma-tags">

</div>

Display the given model.<br />This method will remove the current displaying model, and reset the camera & control to default position.<br />View3D can only show one model at a time

**Returns**: void

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|model|[Model](Model)|||[Model](Model) instance to show|
|options|object|✔️|{}|Display options|
|options.applySize|number|✔️|true|Whether to change model size to given "size" option.|
|options.size|number|✔️|80|Size of the model to show as.|
|options.resetView|boolean|✔️|true|Whether to reset the view to the camera's default pose.|
