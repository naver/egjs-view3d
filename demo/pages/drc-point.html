Draco(.drc) file format as Point Cloud

Draco Github

Use "point: true" and it'll render model as a point cloud

import View3D, {
  DracoLoader, OrbitControl, THREE
} from "@egjs/view3d";

const view3d = new View3D("#view3d-canvas");
const loader = new DracoLoader();

view3d.scene.setBackground(new THREE.Color(0x2288aa));

view3d.controller.add(new OrbitControl());

loader.load("../asset/bunny.drc", { color: 0x999999, point: true, pointOptions: { size: 0.3 } })
  .then(model => {
    view3d.display(model);
  });
