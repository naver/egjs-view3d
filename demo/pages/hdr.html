HDR(RGBE) Background + Envmap

Made with HDR Image from HDRIHaven.com

Scene's background & envmap texture can be loaded with TextureLoader's functions

import View3D, { TextureLoader } from "@egjs/view3d";

const view3d = new View3D("#view3d-canvas");
const textureLoader = new View3D.TextureLoader(view3d.renderer);

textureLoader.loadHDRTexture("../image/moonless_golf_4k.hdr", true)
  .then(texture => {
    view3d.scene.setBackground(texture);
    view3d.scene.setEnvMap(texture);
  }).catch(e => console.error(e));
