Center of the camera rotation.
If `"auto"` is given, it will use the center of the model's bounding box as the pivot.
Else, you can use any world position as the pivot.

## Example
### center: "auto" (default)
<View3D
  src="/egjs-view3d/model/draco/alarm.glb"
  center="auto"
  showExampleCode />

### center: [0.03, 0, 0]
<View3D
  src="/egjs-view3d/model/draco/alarm.glb"
  center={[0.03, 0, 0]}
  showExampleCode />
