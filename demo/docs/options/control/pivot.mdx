Initial focal point of the camera. Also will be referenced while [camera.fit](/docs/api/Camera#fit) is called.
If `"auto"` is given, it will use [center](/docs/options/control/center) as pivot.
Else, you can use any world position.
Use [view3D.camera.pivot](/docs/api/Camera#pivot) instead if you want current pivot value.

## Example
### pivot: "auto" (default)
<View3D
  src="/egjs-view3d/model/draco/alarm.glb"

  showExampleCode />
