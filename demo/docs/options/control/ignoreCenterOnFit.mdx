When [camera.fit](/docs/api/Camera#fit) is called, View3D will adjust camera with the model so that the model is not clipped from any camera rotation by assuming {@link View3D#center center} as origin of the rotation by default.
This will ignore that behavior by forcing model's bbox center as center of the rotation while fitting the camera to the model.

## Example

  <div className="column is-6">

  <div className="column is-6">
