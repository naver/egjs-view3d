Options for the [ZoomControl](/docs/api/ZoomControl).
If `false` is given, it will disable the zoom control.

|Name|Type|Default|Description|
|:---:|:---:|:---:|---|
|scale|number|1|Scale factor for rotation|
|duration|number|300|Duration of the input animation (ms)|
|minFov|number|1|Minimum vertical fov(field of view).<br />You can get a bigger image with the smaller value of this.|
|maxFov|number|"auto"|Maximum vertical fov(field of view).<br />You can get a smaller image with the bigger value of this.<br />If `"auto"` is given, it will use `Math.min(default fov + 45, 175)`|
|easing|function|EASING.EASE_OUT_CUBIC|Easing function of the animation|
