Options for the [ZoomControl](/docs/api/ZoomControl).
If `false` is given, it will disable the zoom control.

|Name|Type|Default|Description|
|:---:|:---:|:---:|---|
|scale|number|1|Scale factor for rotation|
|duration|number|300|Duration of the input animation (ms)|
|minFov|number|1|Minimum vertical fov(field of view).<div className="admonition admonition-caution alert alert--warning"><div className="admonition-heading"><h5><span className="admonition-icon"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 16 16"><path fillRule="evenodd" d="M8.893 1.5c-.183-.31-.52-.5-.887-.5s-.703.19-.886.5L.138 13.499a.98.98 0 0 0 0 1.001c.193.31.53.501.886.501h13.964c.367 0 .704-.19.877-.5a1.03 1.03 0 0 0 .01-1.002L8.893 1.5zm.133 11.497H6.987v-2.003h2.039v2.003zm0-3.004H6.987V5.987h2.039v4.006z"></path></svg></span>caution</h5></div><div className="admonition-content"><p>Only available when type is "fov"</p></div></div>You can get a bigger image with the smaller value of this.|
|maxFov|number|"auto"|Maximum vertical fov(field of view).<div className="admonition admonition-caution alert alert--warning"><div className="admonition-heading"><h5><span className="admonition-icon"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 16 16"><path fillRule="evenodd" d="M8.893 1.5c-.183-.31-.52-.5-.887-.5s-.703.19-.886.5L.138 13.499a.98.98 0 0 0 0 1.001c.193.31.53.501.886.501h13.964c.367 0 .704-.19.877-.5a1.03 1.03 0 0 0 .01-1.002L8.893 1.5zm.133 11.497H6.987v-2.003h2.039v2.003zm0-3.004H6.987V5.987h2.039v4.006z"></path></svg></span>caution</h5></div><div className="admonition-content"><p>Only available when type is "fov"</p></div></div>You can get a smaller image with the bigger value of this.<br />If `"auto"` is given, it will use `Math.min(default fov + 45, 175)`|
|minDistance|number|0.1|Minimum camera distance.<div className="admonition admonition-caution alert alert--warning"><div className="admonition-heading"><h5><span className="admonition-icon"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 16 16"><path fillRule="evenodd" d="M8.893 1.5c-.183-.31-.52-.5-.887-.5s-.703.19-.886.5L.138 13.499a.98.98 0 0 0 0 1.001c.193.31.53.501.886.501h13.964c.367 0 .704-.19.877-.5a1.03 1.03 0 0 0 .01-1.002L8.893 1.5zm.133 11.497H6.987v-2.003h2.039v2.003zm0-3.004H6.987V5.987h2.039v4.006z"></path></svg></span>caution</h5></div><div className="admonition-content"><p>Only available when type is "distance"</p></div></div>This will be scaled to camera's default distance([camera.baseDistance](/docs/api/Camera#baseDistance))|
|maxDistance|number|2|Maximum camera distance.<div className="admonition admonition-caution alert alert--warning"><div className="admonition-heading"><h5><span className="admonition-icon"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 16 16"><path fillRule="evenodd" d="M8.893 1.5c-.183-.31-.52-.5-.887-.5s-.703.19-.886.5L.138 13.499a.98.98 0 0 0 0 1.001c.193.31.53.501.886.501h13.964c.367 0 .704-.19.877-.5a1.03 1.03 0 0 0 .01-1.002L8.893 1.5zm.133 11.497H6.987v-2.003h2.039v2.003zm0-3.004H6.987V5.987h2.039v4.006z"></path></svg></span>caution</h5></div><div className="admonition-content"><p>Only available when type is "distance"</p></div></div>This will be scaled to camera's default distance([camera.baseDistance](/docs/api/Camera#baseDistance))|
|easing|function|EASING.EASE_OUT_CUBIC|Easing function of the animation|
