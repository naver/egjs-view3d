Whether to automatically call [resize()](/docs/api/View3D#resize) when the canvas element's size is changed.
It will use [Resizeobserver](https://developer.mozilla.org/en-US/docs/Web/API/ResizeObserver) for resize detection if `useResizeObserver` set to true, or window's `resize` event if not.

:::info

Change the window size or rotate device to check its effect!

:::
