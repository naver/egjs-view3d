Add plugins before initialization.
Can be removed later by calling [removePlugins](/docs/api/View3D#removePlugins).

```js
import { LoadingBar } from "@egjs/view3d";

const view3D = new View3D("#el_id", {
  plugins: [new LoadingBar()]
})
```
