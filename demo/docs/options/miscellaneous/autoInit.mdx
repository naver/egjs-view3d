Call View3D's [init()](/docs/api/View3D#init) automatically when creating instance.
When the option `src` is not given, `init` won't be called even if this option is true.

If `src` is given, you can call [init()](/docs/api/View3D#init) anytime to initialize View3D.
Or, if the `src` is `null` you can call [load()](/docs/api/View3D#load) to set new 3D model & initialize.
