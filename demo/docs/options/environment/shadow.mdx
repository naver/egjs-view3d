<OptionDescriptor type="boolean | ShadowOptions" defaultVal="true" />

Enable shadow below the model.
If `true` is given, it will enable shadow with the default options.
If `false` is given, it will disable the shadow.
You can also customize the shadow with the following properties.

|Name|Type|Default|Description|
|:---:|:---:|:---:|---|
|opacity|number|0.3|Opacity of the shadow.|
|hardness|number|6|Hardness of the shadow. Should be integer greater than 0, and lower the softer the shadow is.|
|yaw|number|0|Y-axis rotation of the light that casts shadow.|
|pitch|number|0|X-axis rotation of the light that casts shadow.|
