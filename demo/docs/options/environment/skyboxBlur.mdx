<OptionDescriptor type="boolean | number" defaultVal="false" />

Apply blur to the current skybox image.
If `true` is given, it will enable blur with the default blurriness (0.04)
If `false` is given, it will disable the blur.
If a number is given, it will enable blur with the given value as a blur radius in radians. (Higher the blurrier)
