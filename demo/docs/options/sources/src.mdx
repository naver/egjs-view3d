Source URL to fetch 3D model from.

```ts
new View3D("#el", {
  src: "SOURCE_TO_YOUR_3D_MODEL"
})
```

## Example
<View3D src="/model/bunny.glb" preset={new WhiteLightPreset()} />

```ts
import View3D, { WhiteLightPreset } from "@egjs/view3d";

new View3D("#view3d-canvas", {
  src: "/model/bunny.glb",
  preset: new WhiteLightPreset()
})
```
