Sometimes, some rigged model has the wrong bounding box that when displaying on three.js (usually converted glTF model from Sketchfab)
Enabling this option can resolve that issue by recalculating bounding box size with the influence of the skeleton weight.

```ts
new View3D("#el", {
  src: "SOURCE_TO_YOUR_3D_MODEL",
  fixSkinnedBbox: true
})
```

## Example
### fixSkinnedBbox: false
<View3D showBbox src="/model/trex.glb" envmap="/texture/spruit_sunrise_1k.hdr" />

```ts
import View3D, { WhiteLightPreset } from "@egjs/view3d";

new View3D("#view3d-canvas", {
  src: "/model/tyrannosaurus_rex_2.0/scene.gltf",
  envmap: "/texture/spruit_sunrise_1k.hdr"
})
```

### fixSkinnedBbox: true
<View3D showBbox src="/model/trex.glb" envmap="/texture/spruit_sunrise_1k.hdr" fixSkinnedBbox={true} />
