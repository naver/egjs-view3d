Set post-processing of model to bloom effect.

|Name|Type|Default|Description|
|:---:|:---:|:---:|---|
|threshold|number |null|controls the clipping of the effect. Essentially the larger the bloom threshold value, the more the effect will be confined to the brightest pixels in the scene.|
|strength|number |null|bloom effect strength|
|radius|number|null|bloom radius|


## Changing Bloom
You can easily change bloom effect by changing the option value.
