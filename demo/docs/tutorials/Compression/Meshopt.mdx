[Meshopt Compression](https://github.com/KhronosGroup/glTF/blob/main/extensions/2.0/Vendor/EXT_meshopt_compression/README.md) is GPU-friendly mesh optimization that makes meshes smaller and faster to render.
It can dramatically decrease the 3D model size when there're huge amount of geometry data in the 3D model.
i.e. you can apply Draco compression if your glTF model's `.bin` size is big enough to be considered.

Meshopt is not enabled by default in View3D. You can enable it by using the [meshoptPath](/docs/options/model/meshoptPath) option.

[gltfpack](https://github.com/zeux/meshoptimizer/tree/master/gltf) is a great tool to apply `EXT_meshopt_compression` extension. (Use `-c` or `-cc` flag)
