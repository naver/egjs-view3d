[Draco](https://google.github.io/draco/) is a library for compressing and decompressing 3D geometric meshes and point clouds.
It can dramatically decrease the 3D model size when there're huge amount of geometry data in the 3D model.
i.e. you can apply Draco compression if your glTF model's `.bin` size is big enough to be considered.

Draco decoder will be loaded asynchronously with the [dracoPath](/docs/options/model/dracoPath) option.
Note that as Draco decoder size is pretty big (100KB+), so use Draco compression only when file size saving from compression is greater than the decoder size.
By default, Draco decoder will be loaded from their official [CDN](https://www.gstatic.com/draco/versioned/decoders/1.4.1/).

```
https://www.gstatic.com/draco/versioned/decoders/1.4.1/
```

[gltf-pipeline](https://github.com/CesiumGS/gltf-pipeline) is a great tool to apply Draco compression (Use the `-d` flag).
