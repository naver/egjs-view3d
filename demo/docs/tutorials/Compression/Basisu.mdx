View3D supports [KTX v2 texture images with Basis Universal supercompression](https://github.com/KhronosGroup/glTF/blob/main/extensions/2.0/Khronos/KHR_texture_basisu/README.md).

KTX2 texture decoder will be loaded asynchronously with the [ktxPath](/docs/options/model/ktxPath) option.
By default, KTX2 transcoder will be loaded from [unpkg CDN](https://unpkg.com/).

```
https://unpkg.com/three@0.134.0/examples/js/libs/basis/
```

[gltfpack](https://github.com/zeux/meshoptimizer/tree/master/gltf) is a great tool to apply `KHR_texture_basisu` extension. (Use `-tc` flag)
