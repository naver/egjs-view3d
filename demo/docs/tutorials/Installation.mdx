## ðŸ“¦ Package managers (recommended)
You can easily install View3D with package managers like [npm](https://www.npmjs.com/) or [yarn](https://classic.yarnpkg.com/en/)

:::caution
View3D offers different packages for each framework.
For example, you have to install `@egjs/react-view3d` if you're using React.
:::
