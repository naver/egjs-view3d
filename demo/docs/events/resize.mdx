An event that fires when View3D's canvas size is changed.

|Property|Type|Description|
|:---:|:---:|---|
|type|string|The type of event|
|target|[View3D](/docs/api/View3D)|An instance of View3D that triggered this event|
|width|number|New width of the canvas|
|height|number|New height of the canvas|
