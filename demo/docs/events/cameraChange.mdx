An event that fires on every camera movement.

|Property|Type|Description|
|:---:|:---:|---|
|type|string|The type of event|
|target|[View3D](/docs/api/View3D)|An instance of View3D that triggered this event|
|pose|[Pose](/docs/api/Pose)|Current yaw, pitch, zoom, and pivot value|
|prevPose|[Pose](/docs/api/Pose)|Previous yaw, pitch, zoom, and pivot value|
