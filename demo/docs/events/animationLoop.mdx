An event that fires on single animation loop.

|Property|Type|Description|
|:---:|:---:|---|
|type|string|The type of event|
|target|[View3D](/docs/api/View3D)|An instance of View3D that triggered this event|
|index|[Pose](/docs/api/Pose)|An index of the animation|
|clip|[THREE.AnimationClip](https://threejs.org/docs/#api/en/animation/AnimationClip)|An AnimationClip instance of the animation|
|action|[THREE.AnimationAction](https://threejs.org/docs/#api/en/animation/AnimationAction)|An AnimationAction instance of the animation|
