## ARButton

View3D provides a predefined AR button as [Plugin](/docs/api/ARButton).
To use it, you should include CSS/SASS file for it first.

```js
// It's included in the bundle CSS
import "@egjs/view3d/css/view3d-bundle.min.css";

// Or you can use individual css/sass file
import "@egjs/view3d/css/view3d-ar.min.css"
import "@egjs/view3d/sass/view3d-ar.sass"
```

Then, add [ARButton](/docs/api/ARButton) as plugin.

```js
import View3D, { ARButton } from "@egjs/view3d";

const view3D = new View3D("#el", { ... });
view3D.loadPlugins(new ARButton());
```

See [ARButtonOptions](/docs/api/ARButtonOptions) for available options.

### Custom AR Button

You can also create your own custom enter AR button.

```html
<div class="view3d-wrapper">
  <canvas class="view3d-canvas"></canvas>

  <!-- Custom AR Button -->
  <button id="enter-ar">ENTER AR</div>
</div>
```

Add a click handler to the button that calls [view3D.ar.enter()](/docs/api/ARManager#enter)
```js
document
  .querySelector("#enter-ar")
  .addEventListener("click", () => {
    view3D.ar.enter();
  });
```

## AROverlay

Similarly, View3D also provides default AR UI for [WebARSession](/docs/api/WebARSession)
It currently only shows close button at the moment, but more UI elements will be added as default later.

See [AROverlay](/docs/api/AROverlay) and [AROverlayOptions](/docs/api/AROverlayOptions)

```js
import View3D, { AROverlay } from "@egjs/view3d";

const view3D = new View3D("#el", { ... });
view3D.loadPlugins(new AROverlay());
```
